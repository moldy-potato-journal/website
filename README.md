# Moldy Potato

Moldy Potato is a small film journal website for people who have spent too long staring at a paragraph about cinema and realised, with some fatigue, that it was probably never going to make it into a top journal anyway.

So this is the lower journal. Not in the sense that nobody here cares, but in the sense that the usual grand machinery of authority, prestige, and polished confidence has been quietly set aside. The writing can still be serious. The footnotes can still be excessive. The arguments can still be oddly intense. It is just that nobody has to pretend the whole thing arrived from a perfectly funded editorial universe.

The site has two tracks:

- standalone articles
- journal issues

Standalone articles go up on their own. Journal issues gather several pieces together and pretend, as convincingly as possible, that everybody was organised from the beginning.

## Submission

Submissions go through GitHub pull requests to the content repository. If a piece wants to stand alone, it can be proposed as a standalone article. If it belongs to an upcoming issue, it can wait on an issue branch with the other nearly-finished, slightly delayed, and emotionally overcommitted pieces.

## Sections

The main parts of the site are:

- Home
- Articles
- Journals
- Submit
- Status

## Where The Data Lives

The site code lives in this repository.

The publication data lives in the content repository. The main things in that system are:

- `index.json`
- `information.json`
- `articles/`
- `journals/`
